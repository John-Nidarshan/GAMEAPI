Game API – ASP.NET Core Web API with Azure CI/CD

This project is a **cloud-deployed RESTful Web API built using ASP.NET Core and deployed to Azure App Service using a fully automated CI/CD pipeline in Azure DevOps. The API provides CRUD operations for managing game data and demonstrates modern backend development practices including cloud deployment, automated pipelines, and database integration.

🔗 Live API:
https://game-api-john98.azurewebsites.net/swagger/index.html

🚀 Project Overview

The Game API is designed as a production-style backend service that demonstrates:

Building a RESTful Web API
Connecting to a cloud-hosted SQL database
Implementing clean architecture and EF Core
Deploying automatically via CI/CD pipelines
Hosting the application in Microsoft Azure
